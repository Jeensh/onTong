# 공정 관리 기준 (v2)

## 변경 사항 (v1 대비)

-   불량률 목표 강화: 2% → 1.5%
    
-   AI 기반 실시간 불량 감지 추가
    
-   검사 단계 세분화
    

## 공정 검사 기준

-   입고 검사: 외관 + 성분 분석
    
-   공정 중 검사: 실시간 AI 모니터링 + 시간당 샘플링
    
-   중간 검사: 50% 공정 완료 시점 품질 게이트
    
-   최종 검사: 출하 전 전수 검사 + 포장 상태
    

## 불량률 기준

-   목표: 1.5% 이하
    
-   경고: 1.5~3%
    
-   중단: 3% 초과 시 라인 중단 + 원인 분석 필수
    

## 데이터 관리

-   검사 결과 MES 실시간 기록
    
-   월간 품질 리포트 자동 생성
