# 주문 처리 절차

## 개요
고객 주문 접수부터 출하 확정까지의 표준 프로세스를 정의합니다.

## 절차

### 1. 주문 접수
- ERP 주문 모듈에서 신규 주문 생성
- 고객 코드, 품목 코드, 수량, 납기일 입력
- 재고 확인 후 주문 확정

### 2. 생산 의뢰
- 재고 부족 시 MES로 생산 의뢰 전달
- 납기일 기준 역산하여 생산 시작일 산출

### 3. 출하 확정
- 검수 완료된 제품에 대해 출하 지시
- 물류팀에 배송 의뢰

## 주의사항
- 긴급 주문(납기 3일 이내)은 팀장 승인 필요
- 해외 주문은 관세 코드 확인 필수
