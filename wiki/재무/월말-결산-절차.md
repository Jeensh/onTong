# 월말 결산 절차

## 개요
매월 결산 마감 프로세스와 체크리스트를 정의합니다.

## 결산 일정 (D = 익월 기준)

| 일자 | 작업 | 담당 |
|------|------|------|
| D+1 | 매출/매입 전표 마감 | 회계팀 |
| D+2 | 원가 집계 확정 | 원가팀 |
| D+3 | 미결 전표 정리 | 회계팀 |
| D+5 | 월말 재무제표 초안 | 회계팀장 |
| D+7 | CFO 검토 및 확정 | CFO |

## 체크리스트
- [ ] 은행 잔액 조정표 확인
- [ ] 미수금/미지급금 대사
- [ ] 감가상각비 자동 전기 확인
- [ ] 외화 환산 처리
- [ ] 세금계산서 발행 내역 확인

## 주의사항
- D+3까지 미결 전표가 있으면 회계팀장에게 보고
- ERP 마감 처리 후 해당 월 전표 입력 불가
