# 설비보전 점검표

## 일상 점검 (매일)
- [ ] 설비 외관 이상 여부
- [ ] 윤활유 수위 확인
- [ ] 이상 소음/진동 확인
- [ ] 온도/압력 게이지 정상 범위 확인

## 정기 점검 (월 1회)
- [ ] 벨트/체인 마모 상태
- [ ] 필터 교체 여부
- [ ] 전기 배선 점검
- [ ] 안전장치 동작 확인

## 오류 코드

### EQ-001: 과열 경보
- **기준**: 설비 온도 80°C 초과
- **조치**: 즉시 가동 중단 → 냉각 후 원인 분석

### EQ-002: 진동 이상
- **기준**: 진동값 기준치 150% 초과
- **조치**: 가동 중단 → 베어링/축 점검
