# ERP 모듈별 권한 설정

## 개요
ERP 각 모듈(SD, MM, PP, FI, CO)의 사용자 권한 설정 가이드입니다.

## 권한 매트릭스

| 모듈 | 조회 | 입력 | 승인 | 관리자 |
|------|------|------|------|--------|
| SD (영업) | 전체 | 영업팀 | 영업팀장 | IT |
| MM (구매) | 전체 | 구매팀 | 구매팀장 | IT |
| PP (생산) | 전체 | 생산팀 | 공장장 | IT |
| FI (회계) | 재무팀 | 재무팀 | CFO | IT |
| CO (원가) | 재무팀 | 재무팀 | CFO | IT |

## 권한 신청 절차
1. 부서장이 권한 신청서 작성
2. IT 보안팀 검토
3. 최소 권한 원칙에 따라 부여
4. 분기별 권한 재검토
