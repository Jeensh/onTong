# 마스터데이터 관리 지침

## 개요

ERP 마스터데이터(거래처, 품목, BOM 등)의 생성·변경·삭제 기준을 정의합니다.

## 데이터 분류

| 유형 | 예시 | 관리 주체 |
| --- | --- | --- |
| 거래처 마스터 | 고객, 공급업체 | 영업/구매팀 |
| 품목 마스터 | 원자재, 완제품 | 생산기술팀 |
| BOM | 제품 구성 | 생산기술팀 |
| 단가 마스터 | 구매/판매 단가 | 재무팀 |

## 변경 절차

1.  변경 요청서 작성 (ERP 워크플로)
    
2.  데이터 관리자 검토 및 승인
    
3.  변경 실행 및 이력 기록
    
4.  영향도 분석 (BOM 변경 시 필수)
    

## 주의사항

-   품목 코드 삭제 금지 (비활성 처리만 허용)
    
-   단가 변경은 재무팀장 승인 필수
