# ERP 인터페이스 오류 대응

## 주요 오류 코드

### IF-001: 연결 타임아웃
- **원인**: ERP ↔ MES 간 네트워크 불안정
- **조치**: VPN 연결 상태 확인 → 인터페이스 서버 재시작
- **에스컬레이션**: 30분 이내 미복구 시 인프라팀 요청

### IF-002: 데이터 포맷 불일치
- **원인**: 마스터데이터 변경 후 인터페이스 매핑 미갱신
- **조치**: 매핑 테이블 확인 → 변환 규칙 수정 → 재전송
- **예방**: 마스터데이터 변경 시 인터페이스 영향도 점검 필수

### IF-003: 중복 전송
- **원인**: 재처리 시 idempotency key 미설정
- **조치**: 중복 데이터 식별 → 수동 삭제 → 재전송
- **예방**: 모든 인터페이스에 idempotency key 적용
