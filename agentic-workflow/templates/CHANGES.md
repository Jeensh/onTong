# 변경/추가 요청

> 세션 사이에 요구사항이 바뀌거나 추가되면 여기에 메모.
> AI는 매 세션 시작 시 이 파일의 `[ ]` 항목을 확인하고 우선 처리.
> 처리 완료 시 `[x]`로 체크하고 TODO.md에 반영.

---

## 사용법

세션 사이에 요구사항이 생기면 아래 형식으로 추가하세요:

```markdown
## YYYY-MM-DD (세션 N — 주제)

- [ ] **변경 내용** — 구체적 설명
- [ ] **변경 내용** — 구체적 설명
```

AI는 다음 세션 시작 시 `[ ]` 항목을 먼저 확인하고 처리합니다.
처리 완료되면 AI가 `[x]`로 체크합니다.
